Detailed Description

A renderer that handles the multiple values from a MultiValueCategoryDataset by plotting a shape for each value for each given item in the dataset. The example shown here is generated by the

ScatterRendererDemo1.java

program included in the JFreeChart Demo Collection:

Since
1.0.7
