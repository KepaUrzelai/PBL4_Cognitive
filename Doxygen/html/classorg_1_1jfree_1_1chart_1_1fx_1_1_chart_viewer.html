Detailed Description

A control for displaying a JFreeChart in JavaFX (embeds a ChartCanvas, adds drag zooming and provides a popup menu for export to PNG/JPG/SVG and PDF formats). Many behaviours(tooltips, zooming etc) are provided directly by the canvas.

THE API FOR THIS CLASS IS SUBJECT TO CHANGE IN FUTURE RELEASES. This is so that we can incorporate feedback on the (new) JavaFX support in JFreeChart.
