Detailed Description

Internal class to represent a valid segment for this timeline. A segment is valid on a timeline if it is part of its included, excluded or exception segments.

Each segment will know its segment number, segmentStart, segmentEnd and index inside the segment.

boolean org.jfree.chart.axis.SegmentedTimeline.Segment.inExceptionSegments ( )

Returns true if we are an exception segment. This is implemented via a binary search on the exceptionSegments sorted list.

If the segment is not listed as an exception in our list and we have a baseTimeline, a check is performed to see if the segment is inside an excluded segment from our base. If so, it is also considered an exception.
