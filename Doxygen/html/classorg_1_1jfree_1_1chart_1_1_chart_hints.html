Detailed Description

Special rendering hints that can be used internally by JFreeChart or by specialised implementations of the

Graphics2D

API. For example, JFreeSVG's

SVGGraphics2D

class, will use the

KEY_BEGIN_ELEMENT

and

KEY_END_ELEMENT

hints to drive the output content.
