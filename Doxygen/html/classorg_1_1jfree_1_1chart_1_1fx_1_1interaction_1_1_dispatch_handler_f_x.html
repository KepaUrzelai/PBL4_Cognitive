Detailed Description

Handles mouse move and click events on the ChartCanvas by dispatching ChartMouseEventFX events to listeners that are registered with the

ChartCanvas

(listeners can also be registered with a ChartViewer control).

THE API FOR THIS CLASS IS SUBJECT TO CHANGE IN FUTURE RELEASES. This is so that we can incorporate feedback on the (new) JavaFX support in JFreeChart.
