Detailed Description

A category item label generator is an object that can be assigned to a org.jfree.chart.renderer.category.CategoryItemRenderer and that assumes responsibility for creating text items to be used as labels for the items in a org.jfree.chart.plot.CategoryPlot.

To assist with cloning charts, classes that implement this interface should also implement the org.jfree.util.PublicCloneable interface.
