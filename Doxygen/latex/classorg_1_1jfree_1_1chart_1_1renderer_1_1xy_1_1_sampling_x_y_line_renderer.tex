\subsection{Detailed Description}
A renderer that draws line charts. The renderer doesn\textquotesingle{}t necessarily plot every data item -\/ instead, it tries to plot only those data items that make a difference to the visual output (the other data items are skipped). ~\newline
This renderer is designed for use with the \mbox{\hyperlink{}{X\+Y\+Plot}} class.
