\subsection{Detailed Description}
The base class for markers that can be added to plots to highlight a value or range of values. ~\newline
~\newline
 An event notification mechanism was added to this class in \mbox{\hyperlink{classorg_1_1jfree_1_1chart_1_1_j_free_chart}{J\+Free\+Chart}} version 1.\+0.\+3.
