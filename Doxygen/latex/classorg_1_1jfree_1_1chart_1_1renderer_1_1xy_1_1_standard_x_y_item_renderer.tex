\subsection{Detailed Description}
Standard item renderer for an \mbox{\hyperlink{}{X\+Y\+Plot}}. This class can draw (a) shapes at each point, or (b) lines between points, or (c) both shapes and lines. 

This renderer has been retained for historical reasons and, in general, you should use the \mbox{\hyperlink{classorg_1_1jfree_1_1chart_1_1renderer_1_1xy_1_1_x_y_line_and_shape_renderer}{X\+Y\+Line\+And\+Shape\+Renderer}} class instead.
